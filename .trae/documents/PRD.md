## 1. 产品概述

本产品是一款面向市场调研人员的在线问卷调研纯前端应用，帮助用户快速设计、预览问卷并分析回收数据。无需后端服务，所有数据存储于浏览器本地，支持完整的问卷生命周期管理。

- **目标用户**：市场调研人员、产品经理、运营人员
- **核心价值**：零配置快速启动调研，所见即所得的编辑体验，多维度的结果分析
- **使用场景**：用户满意度调研、产品需求收集、市场信息采集、活动反馈收集

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地存储 | 创建、编辑、预览问卷，查看和导出结果 |

### 2.2 功能模块

1. **问卷编辑页**：题目列表、拖拽排序、题目类型选择、属性设置、分组管理
2. **题目库页**：常用题目模板、分类筛选、快速添加到问卷
3. **逻辑设置页**：跳题规则配置、条件分支设置、逻辑可视化
4. **样式预览页**：手机/电脑视图切换、实时预览、答题模拟
5. **结果查看页**：答题统计、图表分析、开放回答列表、数据导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 问卷编辑页 | 题目管理 | 添加单选、多选、填空、评分、排序、分组题；拖拽排序；复制题目；批量修改选项；移动分组 |
| 问卷编辑页 | 属性设置 | 设置必答、选项随机、题目说明、最大/最小选择数 |
| 题目库页 | 模板管理 | 分类展示常用题目模板，支持搜索筛选，一键添加到问卷 |
| 逻辑设置页 | 规则配置 | 设置跳题条件，基于选项答案跳转到指定题目或结束问卷 |
| 样式预览页 | 视图切换 | 手机端/电脑端视图切换，模拟真实答题体验 |
| 结果查看页 | 统计分析 | 展示答题数量、完成率、选项分布图表、开放回答列表 |
| 结果查看页 | 数据导出 | 导出汇总数据为 CSV/Excel 文件 |

## 3. 核心流程

```mermaid
flowchart LR
    A["创建问卷"] --> B["编辑题目"]
    B --> C["设置逻辑"]
    C --> D["预览样式"]
    D --> E["发布问卷（模拟）"]
    E --> F["收集答卷（模拟数据）"]
    F --> G["查看分析结果"]
    G --> H["导出数据"]
```

用户从创建问卷开始，依次完成题目编辑、逻辑设置和样式预览，然后通过模拟数据收集答卷，最终查看分析结果并导出数据。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深邃蓝 `#1e40af`，专业可靠，适合企业级工具
- **辅助色**：活力橙 `#f97316`，用于突出操作按钮和关键提示
- **中性色**：灰色系 `zinc`，提供清晰的视觉层次
- **按钮风格**：圆角矩形，微阴影，悬停时轻微上浮
- **字体**：标题使用 `Playfair Display` 优雅衬线字体，正文使用 `Inter` 清晰无衬线字体
- **布局风格**：左侧导航 + 右侧内容区的两栏布局，卡片式内容容器
- **图标风格**：Lucide 线性图标，保持简洁统一

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|----------|
| 问卷编辑页 | 左侧工具栏 | 题目类型按钮、拖拽手柄、折叠动画 |
| 问卷编辑页 | 中间编辑区 | 题目卡片、选项编辑器、属性面板 |
| 问卷编辑页 | 右侧属性栏 | 题目设置表单、分组选择器 |
| 样式预览页 | 视图切换 | 设备切换标签、预览容器、边框效果 |
| 结果查看页 | 图表区域 | 柱状图、饼图、进度条、数据卡片 |

### 4.3 响应式

- 采用桌面优先设计，主内容区最小宽度 1024px
- 预览页单独实现移动端（375px）和电脑端（100%）视图模拟
- 导航栏在小屏幕下折叠为侧边抽屉
- 触控区域不小于 44px，支持移动端操作

### 4.4 动效设计

- 页面加载：渐入 + 轻微上浮动画
- 拖拽排序：拖动时阴影加深，放置时弹性过渡
- 卡片悬停：轻微上浮 + 边框高亮
- 按钮点击：缩放反馈效果
- 数据加载：骨架屏脉冲动画
